Quick start

You don’t need to install anything. You can use the last version of Canvate directly or you can download the js file and use it in your site.

The last version minified is here.

What is a Clip?

A Clip is any visual object used inside the Canvate.

The Clip acts as button when a Mouse Event is added to it.

It can be used: external images as PNG, JPG, SVG, sprite sheet or the following HTML5 elements:<img>, <canvas>, and <video>.

Inside the Clip (empty or not) it can be added more Clip to manipulate all as one, for complex images and animation construction.

Loading external Image

Inside the Canvate can be loaded external images as: *.jpg; *.png; *.svg etc.

The position, size and rotation can be set before the image is loaded.

An event listener can be added, when the image is loaded: "imageLoaded" and when the image fail to load: "imageError".

// Create the Canvate.
var canvate  = new Canvate("MyCanvas");

// Add new Clip
var maskContainer = canvate.addNewClip();
// Capture image by id
maskContainer.setImageById("Sakura");
// Add an empty Clip inside the
var maskedClip = maskContainer.addNewClip();
maskedClip.setImageById("Fuji");

//Move the masked clip
maskedClip.x -= 55;

//Set the container as mask
maskContainer.setMask();

Using existing image in the page

Masking image

A Clip that contains other clips can act as a mask.

Just create a Clip, add into it other clips, then call the method: setMask.

To remove the mask, just call the method: unsetMask in the Clip container.

See the Pen Masking image (HTML5 canvas) by Eisei Kashi (@EiseiKashi) on CodePen.

Button inside HTML5

All the Clips can be converted easly in a button.

You just need to add an event listener like: "mouseOver", "mouseOut", mouseUp, "mouseOut", mouseDown or "click".

See the Pen Button sample (HTML5 canvas) by Eisei Kashi (@EiseiKashi) on CodePen.
